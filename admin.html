Follow steps 1–5 exactly once to go live on GitHub Pages. After that, managing your site is just: edit here → save → commit to GitHub.
1

Create a GitHub Account (if needed)

Go to github.com → Sign up free. Use a professional username — it appears in your portfolio URL.

2

Create a Public Repository Named "Portfolio"

New Repository → Name: Portfolio → Visibility: Public → Check "Add README" → Create Repository.

3

Upload Both Files

In the repo → Add file → Upload files → drag both:
• index.html — your public portfolio
• admin.html — this CMS (password protected)
→ Click Commit changes.

4

Enable GitHub Pages

Repository → Settings → Pages (sidebar) → Source: Deploy from a branch → Branch: main, Folder: / (root) → Save. Wait 2–3 minutes.

5

Your Live URLs

Portfolio (public): https://sridhar20ch.github.io/Portfolio/
Admin CMS (you only): https://sridhar20ch.github.io/Portfolio/admin.html
⚠️ Bookmark the admin URL.

6

Add Your CV

Upload CV PDF to Google Drive → Right-click → Share → "Anyone with the link" → Copy link → Personal & Links → CV Link field → Save.

7

Future Updates

Edit here → Ctrl+S → changes apply instantly in same browser.
To sync across all devices: After saving → Backup Data (downloads JSON) → Upload both HTML files + JSON backup to your GitHub repo. Changes persist on every device.
